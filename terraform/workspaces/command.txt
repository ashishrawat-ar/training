To create a terraform workspace
# terraform workspace new new_name 

List all the Terraform workspaces
# terraform workspace list 

switch Terraform workspaces
# terraform workspace select name  


create from .tfvars file  (should be on the same workspace)
# terraform plan -var-file="dev.tfvars"
# terraform apply -var-file="dev.tfvars"
# terraform destroy -var-file="dev.tfvars"

this will create seperate workspaces for dev and prod
